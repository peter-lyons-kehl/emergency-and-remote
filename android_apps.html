Do run and test all apps.

Especially so for apps that call themselves "offline". Why? Some "offline" apps need to be run and clicked-through first, so that they download extra resources.

Install apps one by on. Everytime you install one, locate it (by swiping from the bottom and searching), and make a shortcut on your screen pages.

Beware that some apps have icons different between Google Play Store and the local installation! Note (*) below.
